Welcome Home, Screen Print, 2020

This is a screen print using water based ink. The design is based off a song called "Mr Postman" by The Marvelettes. The feelings of waiting for a message or letter from somebody can be overwhelming. Those overwhelming thoughts can cause you to think and do irrational things. The song is based off waiting for a significant other to return home with the uncertainty that they will never return.
